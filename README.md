# Seed Prompt Repository

This repository contains the seed prompt configuration used for AI coding assistants and development workflows.

## Overview

The seed prompt defines the foundational instructions, context, and behaviors for AI assistants working within our development environment. It establishes consistent standards, best practices, and guidelines across projects.

## Files

- **seed-prompt.md** - The main seed prompt configuration file

## Usage

The seed prompt is used to initialize AI coding assistants with:
- Project-specific context and requirements
- Coding standards and conventions
- Workflow preferences and best practices
- Tool usage guidelines

## Maintenance

When updating the seed prompt:
1. Make changes to `seed-prompt.md`
2. Test the changes with your AI assistant
3. Commit with a descriptive message
4. Push to the repository

```bash
git add seed-prompt.md
git commit -m "Update: brief description of changes"
git push
```

## Version History

Changes to the seed prompt are tracked through git commits. Review the commit history to see how the prompt has evolved over time.

## Contributing

When proposing changes:
- Ensure changes align with development standards
- Test thoroughly before committing
- Document the rationale in commit messages
- Keep the prompt clear and maintainable
